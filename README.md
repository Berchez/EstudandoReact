## Project: Hooks react

This project was made to dedicate only to the use of hooks, without reckoning anything graphical.
In this project, the use of:
* React Router Dom
* ESLint, Prettier, editorConfig e Prop-Types
* useState
* useEffect
* useCallback
* useMemo
* useRef
* useContext
* useReducer
